# 사용법

```bash
docker rmi [OPTIONS] IMAGE_ID [IMAGE...]
```

---
# 옵션

| Option       | Short | Description                        |
| :------------: |:-----:| ---------------------------------- |
| `--force`    | `-f`  | 이미지 강제 삭제                   |
| `--no-prune` |       | 태그가 없는 부모 이미지 삭제 안 함 |
