# 기능점수

### 데이터 기능
- 내부논리파일(ILF) : 현재 프로젝트에서 구성한 DB
- 외부연계파일(EIF) : 외부에서 구성한걸 연계한 DB

### 트랜잭션 기능 (CRUD)
- 외부입력(EI) : 사용자 입력
- 외부출력(EO) : 연산이 들어간 출력
- 외부조회(EQ) : 단순 데이터 조회
